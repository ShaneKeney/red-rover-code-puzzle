# Red Rover Code Puzzle

See the following for problem set: https://github.com/RedRoverK12/Red-Rover-Code-Puzzle/blob/main/README.md

## Installing
Ensure you have Node.  This usually comes with Node Package Manager by default.  Then install project dependencies via:

```
npm install
```

## Running project
You can run the project via:

```
npm run dev
```

## Output
By default, `npm run dev` will output the following to the console:
1. First output deliverable
2. First output deliverable using a N-ary tree data structure to temporarily store the data
3. Second outpit deliverable sorted from the stored N-ary data structure

There are also two output files in the `output` dir:
- `output-1.txt`
- `output-2.txt` 

that respectively have the deliverable outputs as specified in the instructions
